Why Notion

  1. Powerful tool

Notion calls itself an all-in-one workspace for notes and documents, wikis, tasks and projects, and spreadsheets and databases. Lots of functions and features in notion make it a powerful tool to do anything you want, it can use as a to-do list, calendar, project manager and etc. If you are familiar with project manager‚ÄìTrello, notion can even do what Trello can do.

Therefore, I want my work manager can run on multiple device and sync data in real-time. And notion can not only have desktop version(macOS, Linux and windows) and mobile version(ios and android), but also can run on website. And this make it convenient for me to use it every time and every where.

  1. Free

And final, notion is free for personal use. I think this is the best advantage for me.

Property

  • Status
    In this properties, I have five tags to choose

Areas on my main page

  • Today

    I put this area on the top of this main page, because the tasks in this area means that these tasks should be done today. And every time I open notion, what I first see is this area, which lets me know if there are still some tasks should I do today.

  • Pomodoro

I follow this way and find that it let me more productivity. When doing some task, it may take you a long time to finish. Therefore, I just split the task into mini task which would takes me about 25 minute to finish it, and I just put these mini tasks into this area. After I done the mini task, I just check it and go through the following task.
